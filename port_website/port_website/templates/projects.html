Health Pet:

Health Pet is an android app that attempts to gamify building healthy routines and habits. Users create name and customize a health pet and set and complete personal goals to unlock points to buy more cosmetic options for their pet with point bonuses and reassurance for achieving streaks with a certain goal. Health Pet was developed in Android Studio using Java, Kotlin, MVVM architecture, and a Firebase database.

Movie Recommendation Algorithm:

Provided users with five movie recommendations based on an inputted movie they enjoy. To do this I used item-item content based to find the top five highest rated movies among other users that rated the inputted movie highly. This was done using the k-nearest neighbors algorithm. The dataset used was the publicly available MovieLens dataset and the algorithm was written in python using sklearn and SciPy.

FEH robot competition:

Designed and built an automated robot to complete an obstacle course within a fixed time limit for an OSU sponsored competition. Our robot’s main mechanism was an arm on a 3D printer like grid that was capable of completing every obstacle without another mechanisms. Robot was controlled with the OSU mobius controller and was programmed in C. Competition was canceled due to covid but robot achieved all goals and bonus goals during each checkpoint prior to the competition.
